# carres-apolliniens
Annexe informatique du projet de recherche

A titre de comparaison, sont proposées sur cette page la version originale du programme python (carres-magiques_original.py) ainsi que sa réécriture (carres-magiques_clean.py).

Il est possible de tester en ligne le programme réécrit à l'adresse https://replit.com/@Nhali/carresapolliniensclean?v=1. Attention cependant, le programme n'est pas prévu pour être exécuté ainsi dans un navigateur : quand la dimension est trop grande (>34), le carré magique obtenu est enregistré dans un fichier à part. Le fonctionnement sera donc optimal si vous téléchargez le fichier pour l'éxecuter sur votre ordinateur personnel.
